# Agrotec Sustentável

## Descrição
Este é um site institucional sobre **Agrotec Sustentável**, mostrando projetos de inovação tecnológica e sustentabilidade no setor agrícola. O site apresenta informações sobre os projetos, a equipe e um formulário de contato.

## Estrutura do Projeto
- **index.html**: Estrutura do site (HTML5 semântica)
- **style.css**: Estilos do site, incluindo responsividade e animações básicas
- **script.js**: Funcionalidades interativas, como scroll suave e alertas de envio de formulário
- **README.md**: Documentação do projeto

## Funcionalidades
- Menu de navegação com scroll suave
- Seção “Projetos” com 4 cards
- Formulário de contato funcional com alerta
- Design responsivo para dispositivos móveis e desktop

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript puro (Vanilla JS)

## Como usar
1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo **index.html** no navegador.
3. Explore o site e teste o formulário de contato.
